# Necesidades, especificaciones y requerimientos del
proyecto.

- [ ]  Explorar y tratar los datos historicos
- [ ]  Generar un modelo de predicción donde el resultado sea explicable
- [ ]  Revisar que las predicciones se ajusten a los datos historicos
- [ ]  Crear una interfaz amigable para personal no técnico
- [ ]  Mostrar el resultado de forma visual
- [ ]  Permitir que el personal introdusca datos nuevos y realizar una predicción en base de esos datos
- [ ]  Que la aplicación sea ejecutable en un entorno estandar, para esto se usará Streamlit
- [ ]  proporcionar un manual tecnico sobre el uso y el mantenimiento de la web
